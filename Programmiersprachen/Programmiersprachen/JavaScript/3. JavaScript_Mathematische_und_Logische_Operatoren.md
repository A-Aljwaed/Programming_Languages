## **5. Typsicherheit in JavaScript**

JavaScript ist eine **dynamisch typisierte** Sprache. Das bedeutet, dass Variablen keinen festen Typ haben, und der Typ einer Variablen sich zur Laufzeit √§ndern kann.

### **5.1 Typumwandlung**
JavaScript f√ºhrt automatisch eine **Typumwandlung** durch, wenn der Typ einer Variablen nicht mit der erwarteten Operation √ºbereinstimmt.

- **Beispiel**:
  ```javascript
  let zahl = 10;
  let text = "10";
  console.log(zahl == text); // true (lose Gleichheit, Typen werden konvertiert)
  console.log(zahl === text); // false (strikte Gleichheit, Typen sind unterschiedlich)
  ```

### **5.2 Explizite Typumwandlung**
Man kann auch explizit einen Typ umwandeln, z. B. mit `String()`, `Number()` oder `Boolean()`.

- **Beispiel**:
  ```javascript
  let num = 10;
  let numStr = String(num); // Umwandlung von Zahl in String
  console.log(typeof numStr); // "string"
  ```
